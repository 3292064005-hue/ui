# Operation Flow

1. Connect robot and peripheral devices.
2. Power on, switch to automatic mode, validate tool/load/TCP.
3. Create experiment and lock the configuration snapshot.
4. Run localization and generate the scan trajectory.
5. Move to pre-scan pose with non-real-time motion.
6. Enter RT contact mode and establish stable contact.
7. Run segmented scan with quality monitoring.
8. Save synchronized data packages.
9. Run preprocessing, reconstruction and assessment.
10. Export summary and report.
